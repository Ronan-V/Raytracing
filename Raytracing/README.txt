Execution du programme de génération d'image.

1 - Editer le fichier de configuration "xxx.txt"


	1.1 - Choisir le mode			 ->	2 modes sont disponibles : sans perspective (=0), avec perspective (=1)

	1.2 - Ajouter des objets à la scène      ->	sphere ou plan. Definir la sphere avec ses coordonnées (x,y,z).
							Definir le plan avec un point du plan et un vecteur normal au plan.

	CONSEILS D'UTILISATION: Definir les coordonnées de la sphere dans cet interval : x = [-350 : 350], y = [-250 : 250], z = [-7 : 200]
				Definir les coordonnées du plan dans cet interval : point = [(-500:500), (-500:500) , (-500,500)], vecteur normal = [(0:1), (0:1) , (0:1)]
				Le premier objet créé est regardé par la caméra.

2 - Double cliquer sur la solution "Raytracing.exe" pour lancer le programme.
